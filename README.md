A WordPress plugin for telecommunications companies, tower companies, and real estate firms managing cell tower lease portfolios. == Description ==

A WordPress plugin for telecommunications companies, tower companies, and real estate firms managing cell tower lease portfolios. Tracks lease agreements with rent escalation schedules and option periods, monitors site maintenance work orders with contractor management, calculates monthly revenue per site and tower, sends automated renewal negotiation reminders 12 months before expiration, manages co-location agreements for multiple tenants per tower, and generates portfolio summary reports for investor reporting.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Telecommunications Tower Lease and Site Management WordPress Plugin
